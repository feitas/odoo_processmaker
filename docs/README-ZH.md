# ODOO链接ProcessMaker

B.P.M. 将 <a href="http://www.odoo.com">Odoo</a> (15) 与 <a href="https://www.processmaker.com/">ProcessMaker</a> (4)进行了对接。

Odoo16即将发布，届时将会创建16.0分支，欢迎大家贡献代码。
