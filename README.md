# ODOO链接ProcessMaker

B.P.M. 将 <a href="http://www.odoo.com">Odoo</a> 与 <a href="https://www.processmaker.com/">ProcessMaker</a> (4)进行了对接。

本仓库用于Odoo的14、16版本，因为重构所以不再fork之前的仓库。
